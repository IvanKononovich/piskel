Piskel is a free online editor for animated sprites & pixel art

Create animations in your browser.

Create Sprite

Example sprites

In the editor implemented:

Pen - implements per-pixel drawing

Eraser - allows you to make the pixels transparent

Paint bucket - allows you to fill the pixels with the selected color

Dithering - allows you to create a checkered pattern

Lighten - allows to lighten and darken colors

Move - allows you to shift the picture on canvas

Rectangle - allows you draw rectangles

Stroke - allows you draw stroke

Pipette color - allows you to choose the color of a particular pixel

Frames - it is possible to create, delete, copy and change the sequence of frames

Layers - you can create, delete, move and merge layers

Resize canvas - it is possible to manually set the width and height of the canvas

Uploading a file to the PC - it is possible to upload the created animation to your PC in gif format

Resize tool - it is possible to choose the size of the tool

Preview - it is possible to view the created animation before you download the giff to your PC, you can choose the frame playback speed, as well as you can turn on the full-screen animation view
